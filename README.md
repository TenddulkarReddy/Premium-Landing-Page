# Task 1: Premium Responsive Landing Page

A modern, responsive landing page built as part of my web development tasks. This project features a clean, professional user interface with advanced navigation controls.

---

## 🚀 Features
* **Premium Glassmorphism Navbar:** A sleek, frosted-glass top navigation bar that styles modern layouts.
* **Active Scroll-Spy Tracker:** Automatically highlights the current page section in the navigation menu as you scroll.
* **Fully Responsive Design:** Adapts smoothly across mobile, tablet, and desktop screens with a slide-out hamburger menu.
* **Engaging Call to Action:** Features a targeted "Get Started" button designed to drive user engagement.

---

## 🛠️ Tech Stack Used
* **HTML5:** Clean, semantic markup structure (`<nav>`, `<section>`, `<header>`).
* **CSS3:** Custom layout styling, glassmorphism design effects, and media queries for total responsiveness.
* **JavaScript (ES6):** Logic handling for the mobile interactive toggle menu and active scroll tracking.
